Lab 1

So, what is programming anyways? Let's learn about how we can represent equations and other familiar objects in code using variables and types. Then, let's learn about arithmetic using unary, binary and comparison operators. We learn about how we can store multiple pieces of data using lists, and end with how we can use methods to manipulate lists and strings.

Lab 2

Flow is the navigation through various choices in a program, and loops allow us to repeat operations. We use these techniques to further our capabilities in Python. We also talk about the mutability of data types: whether variables may be changed, and the implications of doing so. Finally, we look at how multidimensional lists can represent matrices and tensors.

    Lab 3

    Functions! What are they? How are they different from functions in math? What is recursion?

    Lab 4

    What are namespaces, and when can we reference variables in a function? Which data types are passed to functions by value and which are passed by reference? We also investigate list comprehensions, both single and multidimensional.

    Lab 5

    Let's perform calculus in code. We represent polynomials using lists, and create automatic differentiation using numerical methods. We start learning about modules including math, time, numpy, matplotlib and sympy. We also find roots of equations using the grid method, bisection method, and Newton's method.

    Lab 6

    We expand our knowledge of data types by considering sets and tuples.

    Lab 7

    A dictionary isn't just a type of book!

    Lab 8

    Comprehensions and dictionaries go together. Also, we return to coding calculus by considering multiple differentiation and Taylor Series.

    Lab 9

    We return to numpy arrays in depth and also look at the random library.
