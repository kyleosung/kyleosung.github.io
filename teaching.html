Review Sessions

I'm the VP Academic of the McMaster Math and Stats Society. We run Review Sessions and Mock Midterms for first year students to help study and practice. Here are some of the review sessions I've run:

Math 1B03 - Linear Algebra 1

  • Test 1 Review Session (February 3, 2023)
    • 50 Attendees. Linear systems, dependence, transformations, row reduction.

  • Test 2 Review Session (March 22, 2023)
    • 30 Attendees. Vector spaces, subspaces, bases, dimension. Determinants.

Math 1MP3 - Intro to Scientific Computation (Python)

  • Test 1 Review Session (February 10, 2023)
    • 40 Attendees. Intro to Python, variables, strings and methods, lists, functions.

  • Test 2 Review Session (March 16, 2023)
    • 40 Attendees. Data Structures: Lists, sets, dictionaries. Functions and methods

  • Test 2 Review Session (March 17, 2023)
    • 70 Attendees. Data Structures: Lists, sets, dictionaries. Functions and methods

    At the request of the instructor, we chose to run two review sessions for the second test of Math 1MP3 due to timetable conflicts with exams in other courses. I'm happy that this led to one of the largest turnouts I had had thus far for a review session.
